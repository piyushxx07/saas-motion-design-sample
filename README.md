# SaaS Video Learning Roadmap

A clean, animated timeline webpage showcasing a four‑part SaaS learning roadmap (Oct 2025 → Sep 2026). Each milestone features a short video (Saas_1.mp4 – Saas_4.mp4) that plays automatically and pauses others when played.

![Screenshot of the webpage](Motion_Design\Screenshort.png)

- [Saas_4.mp4](Saas_4.mp4) – Advanced topics & Series A prep (Sep 2026)

## How to Run
1. Clone or download this repository.  
2. Ensure the four `.mp4` files and `screenshot.png` are in the same folder as `index.html`.  
3. Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge).  
4. The first video will start muted; click any video to play it and it will pause the others.
